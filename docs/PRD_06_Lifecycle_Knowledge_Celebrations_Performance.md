# PRD 06 — Employee Lifecycle, Knowledge Bank, Celebrations & Performance

## Employee Lifecycle Management

### Onboarding Module

**Pre-Boarding (Before Start Date):**
- Welcome email dispatch
- Policy and handbook sharing
- Form completion workflows
- Task checklist (IT setup, Laptop, Workspace, System access)
- Automated reminders

**First Day Experience:**
- IT setup, Email/software access, Office tour
- Team introductions, Policy orientation, HR documentation
- Buddy/mentor assignment

**First Week Integration:**
- Orientation scheduling, Key stakeholder meetings
- Check-ins, 30-day goals, 90-day goals

**Probation Management:**
- Standard 3-month probation period
- Review reminders, Appraisal scheduling, Confirmation decisions

**Progress Tracking Dashboard:**
- Overall onboarding completion percentage
- Outstanding tasks, Documentation status, Timeline progress

### Offboarding Module

**Exit Process Triggers:**
Resignation, Termination, Retirement, Contract expiration

**Automated Exit Checklist:**
- Asset return (Laptop, Phone), System access revocation
- Handover completion, Final payroll processing
- Exit interview scheduling

**Final Settlement:**
- Final salary, Loan deductions, Final payslip

**Offboarding Records:**
- Exit date, Exit reason, Interview notes, Asset return status

## Policy, Handbook & Knowledge Management (Central Knowledge Bank)

### Core Functions
- Central document repository for policies, handbooks, SOPs, compliance docs
- Folder structure & categorization (HR Policies, Compliance, SOPs, Legal, Templates)
- Searchable, tagged, role-based visibility, archived folders

### Version Control
- Maintain version history for every document
- Track upload date, revision date, author, approval owner
- Compare/restore previous versions

### Access Control
- **Super Admin:** Full access, upload/edit/delete, approve final versions
- **HR Admin:** Upload HR policies, publish resources, track acknowledgments
- **Managers:** Access department resources, upload SOPs (with approval)
- **Employees:** View approved materials, download, acknowledge, search

### Policy Acknowledgment
- Employees notified on new/updated policies
- Digital acknowledgment required
- Completion deadlines, automated reminders
- Dashboard: viewed, acknowledged, outstanding, compliance % by department

### Search & Discovery
Search by: document title, keyword, category, department, date updated.
Features: suggested resources, frequently accessed, recently updated, pinned docs, FAQ

## Performance Management

### Monthly Goals
- Employees set OKRs/goals yearly and monthly
- Goals should be SMART
- Manager reviews and approves goals
- Goals align with department/company objectives

**Goal Structure:**
- Goal title, Detailed description, Success criteria
- Weight assigned (total 100), Priority (High/Medium/Low)
- Target completion date

**Goal Templates:**
- Common templates for roles
- Ensure consistency

### Monthly Check-ins (One-on-Ones)

**Scheduling:**
- Recurring monthly check-ins
- Automatic calendar invites, reminder notifications

**Team Member Self-Appraisal:**
- Rates self, inputs comments and challenges

**Manager Rating:**
- Rates team member against set goals
- Rate each goal over 100, total rating comes to 100

**Check-in Agenda:**
- Review previous month's goals and scores
- Current month progress, challenges, well-being check

**Documentation:**
- Record key discussion points
- Note action items, track commitments
- Keep history of all check-ins

**Follow-up:**
- Track action items to completion
- Remind about pending actions

### Manager Dashboard
- See all upcoming check-ins
- View overdue check-ins
- Track team member engagement
